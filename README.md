
# PokeHead

A simple Pokemon database

This is an example Android project




## Tech Stack

- Kotlin
- Jetpack Compose
- Room
- Retrofit
- Hilt


## Architecture patterns

- CLEAN
- MVVM
- Unidirectional Data Flow
- Use Cases
